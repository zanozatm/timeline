У таймлайна есть несколько состояний:

  1. .timeline__step - шаг не пройден - по умолчанию для элемента списка;
  2. .timeline__step-progress - шаг в процессе - дополнительный класс к элементу списка с классом timeline__step;
  3. .timeline__step-active - шаг пройден -  дополнительный класс к элементу списка с классом timeline__step.

По умолчанию для компонента заданы опции:

  • лейблы по центру прогресс-линии,
  • лейблы над таймлайном.

Чтобы изменить положение лэйблов, к элементу списка добавьте классы для отображения:

  • .timeline__label-onbottom  - лейблов под таймлайном;
  • .timeline__label-onpoint - лейблов по центру чекпоинта. 

Добавьте оба класса для отображения лэйблов под таймлайном и по центру чекпоинта.
